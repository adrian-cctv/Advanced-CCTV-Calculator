Enterprise Storage &amp; Design Utilities for the Security Industry. RAID-aware calculations with binary overhead correction.


## 🛠️ The Tool: Storage Architect v1.0
A high-precision calculator for enterprise CCTV retention.

Key Features
Binary Accuracy: Corrects the 9.1% decimal-to-binary drive gap.

Chassis Visualizer: Real-time topology preview for server bay planning.

VMS Overhead: Includes 5% buffer for indexing and metadata.

How to Use
Enter your total usable TB required.

Select your RAID level (RAID 6 recommended for 16TB+ drives).

Visualise the chassis layout to ensure expansion readiness.
